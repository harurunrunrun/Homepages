Problem Statement

Ursula and Bob are talking about a programing problem.


Ursula:「Submit your code about this programing problem!」

Bob    :「Ursula, I don't know what should I do!」

Ursula:「Bob, did you read the programing problem? You can understand easily!」

Bob    :「Mm...I can't understand.」

Ursula:「I'm surprised.I can't understand you don't understand such a easy programing problem.」

Bob    :「Tell me, Ursula.」

Ursula:「In this programing problem, the input is string $S$ consisting of alphabet. You have to submit input. Do you understand?」

Bob    :「No problem. I'm submitting now. ...What? I can't get AC!」

Ursula:「Please try reading the programing problem again.」

Bob    :「Ursula, I can't understand.」

Ursula:「Think yourself.I can't tell you about the programing problem more.」


Please solve this programing problem instead of Bob.




Input

$S$

The string $S$ is given on the first line.

Constraints

$S$ consists of alphabet (including both lowercase and uppercase letters).

$1≤|S|≤10$ ($|S|$ means $S$ 's size.)

Output

Print the answer on one line.

Please start a new line at the end of the answer.
